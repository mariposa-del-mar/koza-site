# Коза и нейросети

Сайт-визитка для эко-проекта «Коза и нейросети» — экобизнес, ремесленничество, натуральные продукты.

## Структура проекта

```
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
│   ├── icons/
│   │   └── favicon.svg
│   └── images/
└── README.md
```

## Особенности

- Чистый HTML/CSS/JS без фреймворков
- Мобильность (responsive) от 320px до 1920px
- Плавные scroll-анимации (Intersection Observer)
- Липкая навигация, карусель отзывов, FAQ-аккордеон
- Форма подписки с валидацией
- SEO-оптимизация, Open Graph теги
- Доступность (a11y): alt-теги, aria, focus-состояния

## Как открыть локально

1. Откройте папку проекта
2. Откройте `index.html` в браузере (двойной клик)

Или через терминал:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve .
```

Затем откройте `http://localhost:8000`
